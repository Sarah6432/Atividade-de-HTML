A História dos computadores, como tudo começou?

voltar

Que os computadores são equipamentos muito úteis e facilitam nosso dia a dia você já sabe, mas você já se perguntou quando surgiu o primeiro computador, quem o criou e qual era seu nome? Hoje vou explicar um pouco sobre como essas máquinas surgiram.

O primeiro computador surgiu entre 1943 e 1946 e ele se chamava Eletronic Numerical Integrator and Computer mais conhecido como ENIAC.

O projeto do ENIAC teve seu início em 1943, durante a Segunda Guerra Mundial, o objetivo principal era criar uma máquina capaz de realizar cálculos para a artilharia dos Estados Unidos, foi projetado e construído por Jonh W. Mauchly e J. Presper Eckert na Universidade da Pensilvânia. O ENIAC foi concluído em 1945 e se tornou operacional em novembro desse mesmo ano. Foi anunciado ao mundo em fevereiro de 1946.

Logo após o ENIAC surgiram o EDIVAC e UNIVAC que foram as evoluções do ENIAC.

O desenvolvimento do EDIVAC começou em 1944, enquanto o ENIAC estava sendo construído.O Eletronic Discrete Variable Automatic Computer - EDVAC foi um dos primeiros computadores eletrônicos.

Ao contrário do ENIAC, não era decimal, mas binário e foi o primeiro programa concebido para ser armazenado.

O UNIVAC - Universal Automatic Computer - 1951, era programado ajustando-se cerca de 6.000 chaves e conectando-se a um painel.

O primeiro UNIVAC foi entregue ao escritório do censo dos Estados Unidos em 31 de março de 1951, mas demorou para começar a funcionar, então o primeiro que entrou em operação foi o segundo a ser fabricado, para o Pentágono.

Assim se iniciou a história dos computadores, essa foi a primeira geração.

A segunda geração (1956-1963) : Com a invenção do transistor, os computadores se tornaram menores, mais rápidos e mais eficientes em termos de energia. Essa geração introduziu o uso de linguagens de programação de alto nível, como COBOL e FORTRAN.

1960s - Computadores de Terceira Geração : Circuitos Integrados: Introduzidos nos anos 1960, permitiram a miniaturização e aumento da eficiência dos computadores. IBM System/360: Introduzido em 1964, foi um marco por sua arquitetura compatível, permitindo a atualização de hardware e software sem a necessidade de reescrever programas.

1970s - Computadores de Quarta Geração : Microprocessadores: A invenção do microprocessador, um circuito integrado que contém todas as funções de uma CPU, foi um avanço crucial. O Intel 4004, lançado em 1971, foi o primeiro microprocessador comercial. Computadores Pessoais (PCs): O Altair 8800, lançado em 1975, é considerado o primeiro computador pessoal, e foi seguido pelo Apple II em 1977.

1980s e 1990s - Popularização dos PCs e a Era da Internet : IBM PC: Lançado em 1981, tornou-se o padrão para computadores pessoais. Apple Macintosh: Introduzido em 1984, foi pioneiro na interface gráfica com o usuário (GUI). Internet: A rede ARPANET, precursora da Internet, foi desenvolvida nos anos 1960, mas a World Wide Web, criada por Tim Berners-Lee em 1989, popularizou a Internet nos anos 1990.

O que são Clients?

voltar

Os clients são os usuários, os dispositivos que solicitam serviços ou recursos a servidores que são dispositivos que recebem e processam as solicitações e fornecem uma resposta adequada.

O que são Servers?

voltar

Os servers são recursos dentro de sistemas computacionais, capazes de processar aplicações, armazenar dados e prestar serviços. Enquanto o cliente é um computador que solicita esses serviços e recursos. Em uma infraestrutura de TI, ambos formam a arquitetura conhecida como "cliente-servidor".

O que é HTML?

voltar

O HTML - A Linguagem de Marcação de Hipertexto, é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento. O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

Como é uma estrutura básica em HTML?

voltar

A estrutura básica de um documento HTML inclui as tags <'html'>, <'head'> e <'body'>. A tag <'html'> define o início e o fim do documento HTML. A tag<'head'> contém informações sobre o documento, como o título da página e links para estilos e scripts.Elas são usadas para iniciar o escopo do HTML, através delas você consegue compor a base de todo texto HTML.

O que são as tags em HTML?

voltar

As tags HTML são elementos usados ​​para marcar e formatar os conteúdos: textual ou visual em documentos HTML. Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.Atualmente, existem 142 tags HTML disponíveis, que permitem a criação de diversos elementos.

Exemplos de tags e para que servem:

  • Tags de Cabeçalho – elas variam entre <'h1'> e <'h6'>, onde h1 tem um tamanho maior, que vai diminuindo até chegar no h6.
  • Tags de parágrafo – são abertas usando a tag <'p'>.
  • Tags de lista – possuem diferentes variações.Use a tag <'ol'> para uma lista ordenada, e a tag <'ul'> para uma lista sem ordem definida. Então, envolva cada item da lista com a tag <'li'>.

    • Diferença entre listas ordenadas e não ordenadas:

    voltar

    Em HTML, as listas são uma maneira de agrupar itens relacionados. Existem dois tipos principais de listas: ordenadas e não ordenadas. A diferença entre elas é bem simples e está relacionada à maneira como os itens são apresentados.

    Lista Ordenada (<'ol'>):

    Uma lista ordenada é usada quando a ordem dos itens é importante. Cada item é numerado automaticamente pelo navegador, o que é útil quando você precisa seguir uma sequência específica, como uma receita, um passo a passo, ou uma lista de tarefas.

    Lista Não Ordenada (<'ul'>):

    Uma lista não ordenada é usada quando a ordem dos itens não é importante. Em vez de números, os itens são marcados com pontos (bullets), o que é ideal para listar coisas que não têm uma sequência específica, como ingredientes, características de um produto, ou uma lista de verificação.
